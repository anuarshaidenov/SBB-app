## Getting Started

- Getting a local copy is simple. Follow the steps below

### Install

- Ruby
- Ruby on Rails
- PostgresSQL
- Rspec

### Local Copy

- Clone the project

```
git clone https://github.com/anuarshaidenov/SBB-app.git

cd SBB-app

```

### Setup

Install gems:

```
bundle install
```

Setup database:

```
rails db:create
rails db:migrate
```

### Usage

Start server:

```
rails server
```

Visit http://localhost:3000/ in your browser.

### Run tests

```
rspec
```
